From Website to Dashboard: NCAA Referee Travel & Workload Analysis

Goal: Explore whether referee travel distance influences officiating behavior during NCAA Division I Men’s Basketball games.

Problem Statement

March Madness brings excitement—but is it always fair? I explored whether referee travel distance impacts foul calls. To answer that, I had to build the entire data pipeline from scratch, beginning with zero structured access.

Building the Dataset

The official stats.ncaa.org site has no API—just static HTML pages. Every game ID had to be scraped manually by navigating through dated URLs. I created a system to automate daily scraping across the 2024–25 season, extracting team matchups, venues, and referee assignments. Each row of the final dataset became a fully contextualized game entry.

  • Games indexed: 5,922
  • Columns: 34 features including venue, foul types, referees
  • Referees tracked: 778

Mapping Referee Travel

I used OpenStreetMap’s Nominatim geocoder to convert venue names into coordinates, then calculated straight-line (great-circle) distances between consecutive game locations for each referee. This allowed me to estimate total travel mileage, venue diversity, and rest windows across the season.
